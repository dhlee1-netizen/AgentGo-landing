FAQ

자주 묻는 질문

데이터가 외부로 나가지 않나요?▾
온프레·폐쇄망 배포를 지원합니다. Guard가 접근을 암호화하고 감사 로그를 기록합니다.
AI 오류 시 책임은?▾
전체 감사 로그로 지시·행동·시점을 추적합니다. 원인과 책임을 명확히 파악할 수 있습니다.
기존 시스템과 연동되나요?▾
MCP 표준으로 ERP·그룹웨어 등과 연동됩니다. 기존 인프라 변경 없이 도입 가능합니다.
ROI는 어떻게 증명하나요?▾
Management 대시보드에서 사용량·처리 건수·절감 시간을 실시간 확인합니다.
